
## **Features**

- Fetches balance sheet data from a backend API.
- Displays balance sheet data with proper formatting.
- Includes loading and error states for API calls.
- Tested with Jest

  ## **Prerequisites**

Ensure you have the following installed:

- **Node.js** (v16+): [Download Node.js](https://nodejs.org/)
- **npm** or **yarn**: Node Package Manager
- **Git** (optional, for cloning the repository)

## **Installation**

   **Clone the repository**:
   ```bash
   git clone git@github.com:CoolDev198/xero-frontend.git

   cd xero_balance_sheet_frontend

   npm install
   **Local api server should run as below**:

  http://127.0.0.1:8000/balance-sheet

   **Run tests**:
npm test
